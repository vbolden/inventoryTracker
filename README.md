## OUTLINE

Create a simple object-oriented inventory tracker using TypeScript. The tracker will distiniguish between PhysicalProduct and DigitalProduct, calculate applicable taxes, and manage inventory using modules.

# CRITICAL THINKING 

1. TypeScript enforces type safety by being more strict about the types assigned to different values and it tracks that value throughout the code even in different modules.

2. Inheritance made it so that I didn't have to rewrite the values for name, sku, and price for each module and I didn't have to rewrite them in the constructor either, which made it faster to create each of them. 

3. It helps with maintainability by allowing internal code changes without affecting external users and prevents unauthorized data manipulation and corruption. 

4. Polymorphism would make this extension straightforward because the logic for product already exists so you would just have to create the new type to fit into the existing class shape and on any needed properties for this new type.
